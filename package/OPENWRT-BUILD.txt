Layout (this directory)
-----------------------
  version.mk                    — PKG_VERSION / PKG_RELEASE for both Makefiles
  glinet-privacy/Makefile       — shell, UCI, firewall helpers (no LuCI)
  luci-app-glinet-privacy/Makefile — LuCI JS views (htdocs/…/view/glinet_privacy/*.js), rpcd ACL, menu.d JSON

Feed integration
----------------
See ../feeds.conf.example (src-git …;subdir=package or src-link …/package).

After feeds install, enable in menuconfig:
  * Network → glinet-privacy
  * LuCI → Applications → luci-app-glinet-privacy

Build examples (OpenWrt buildroot):
  make package/glinet-privacy/compile V=s
  make package/luci-app-glinet-privacy/compile V=s

Notes
-----
* Tor / wireguard kmod are not hard Depends on glinet-privacy (optional at runtime); install.sh pulls more than the ipk metadata.
* Translations: po/ + Makefile Build/Compile runs po2lmo (PKG_BUILD_DEPENDS:=luci-base/host); .lmo install to usr/lib/lua/luci/i18n/. See luci-app-glinet-privacy/po/README (Weblate-oriented).
* remove.sh --opkg assumes package names glinet-privacy and luci-app-glinet-privacy.
