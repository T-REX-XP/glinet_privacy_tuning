Privacy router (GL-XE300 Puli, GL-AXT1800 Slate AX / OpenWrt)
==============================================================

All steps below are automated by install.sh unless you use --minimal.

One command (replace USER/REPO; add --with-luci for LuCI files):

  curl -fsSL https://raw.githubusercontent.com/USER/REPO/main/install.sh | \
    GLINET_PRIVACY_TARBALL_URL=https://github.com/USER/REPO/archive/refs/heads/main.tar.gz sh -s -- --with-luci

From a git clone on the router:

  sh install.sh --with-luci

To remove (from the same clone, or set GLINET_PRIVACY_SRC to the repo root):

  sh remove.sh
  sh remove.sh --opkg    # if you installed luci-app-glinet-privacy / glinet-privacy via opkg

See remove.sh --help. Does not remove tor, wireguard-tools, etc., unless you use opkg for our ipks.

What install.sh does by default (not --minimal)
-----------------------------------------------
- Copies package/glinet-privacy/files to /, chmod scripts, installs /usr/share/glinet-privacy/version.mk
- Registers firewall.glinet_privacy → fw-plugin.sh; runs apply-device-profile.sh (WAN per device)
- opkg update + install: iptables, iptables-mod-*, wireguard-tools, kmod-wireguard, tor, tor-fw-helper,
  dnsmasq-full, ca-bundle (best-effort if offline)
- Tor: ensures %include /etc/tor/torrc.d/99-transparent.conf in /etc/tor/torrc; enables and starts tor
- Kill switch: enables /etc/init.d/privacy-killswitch; adds cron */1 for privacy-killswitch-watchdog.sh
- Telemetry: uci glinet_privacy.tel.block_domains=1 and disable_vendor_cloud=1; apply-telemetry.sh
  (dnsmasq.d black-holes for goodcloud.xyz / gldns.com, glconfig.cloud.enable=0, gl_cloud stop/disable);
  optional remove_cloud_packages removes gl-cloud via opkg; dhcp dnsmasq confdir /etc/dnsmasq.d when possible
- Restarts network, firewall, dnsmasq, rpcd, uhttpd

Optional flags
--------------
  --minimal           Files + firewall + device profile only (no opkg/Tor/cron/telemetry automation)
  --with-luci         Install LuCI app (Services → GL.iNet Privacy)
  --with-imei-boot    Enable rotate_imei on boot (cellular; read docs/devices.md IMEI section)
  --with-imei-cron    Add cron: IMEI rotate every 6 hours (optional ROTATE_IMEI_SUPPRESS_LEGAL_LOG=1 in crontab)

VPN (WireGuard / OpenVPN)
-------------------------
Configure the client in the GL.iNet admin. Then set the WireGuard interface name in LuCI
(Services → GL.iNet Privacy → Kill switch) so the watchdog matches your tunnel.

Prebuilt .ipk (optional)
-------------------------
See package/BUILDING.txt and GitHub Actions artifacts / releases.

Notes
-----
- IMEI: legal use and operator/lab framing — **docs/devices.md** (IMEI rotation section); each run logs a notice unless `ROTATE_IMEI_SUPPRESS_LEGAL_LOG=1`.
- DNS: LuCI **Tor, DNS & telemetry** (`glinet_privacy.dns`) or **`apply-dns-policy.sh`** (`default` or `tor_dnsmasq`).
- Tor + VPN: use one routing path you understand; policy routing is outside this package.
- docs/devices.md — WAN defaults per model (Puli vs Slate AX).
